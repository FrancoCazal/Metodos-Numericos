# Métodos Numéricos - F.I.U.N.A.

### Descripción:
Este repositorio contiene clases y ejercicios de la cátedra de Métodos numéricos de la Faculta ded Ingeniería de la Universidad Nacional de Asunción.

### Objetivo:
Se busca que sirva como referencia para uso durante la cursada de la materia como también para referencia futura.
